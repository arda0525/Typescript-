# Typescript-

1-mergeArrays: İki diziyi birleştirip tek dizi yapar.

2-useVehicle: Aracın araba mı kamyon mu olduğunu kontrol eder.

3-Logger: Logları kaydeder ama dışarıdan erişilmesini engeller.

4-getProperty: Nesneden istenen özelliği alır.

5-search: ID ile tek kullanıcıyı, isimle benzer kullanıcıları döndürür.

6-MemoryCache: Veriyi kaydeder, okur ve temizler.

7-updateUser: Kullanıcı bilgilerini günceller ve değiştirilmesini engeller.

8-sum: Verilen tüm sayıları toplayıp sonucu verir.

9-Shape/Circle: Dairenin alanını hesaplayan sınıf oluşturur.

10-MathHelper: PI sabiti ve çevre hesaplama fonksiyonunu içerir.

11-UnwrapPromise: Promise içindeki değeri alır, yoksa olduğu gibi döndürür.

12-GetReturnType: Fonksiyonun dönüş tipini bulur.

13-CreateGetters: Nesnedeki özellikleri “get” fonksiyonuna dönüştürür.

14-DeepReadonly: İç içe tüm nesneleri salt okunur yapar.

15-PickByValueType: Belirli türdeki değerleri seçer.

16-Branded Types: Aynı tipte ama karışmayan özel türler oluşturur.

17-FilterUnion: Union içinden istenen türleri çıkarır.

18-LastParameter: Fonksiyonun son parametresinin tipini bulur.

19-Flatten: Dizi içindeki eleman tipini alır.

20-ParseRouteParams: /users/:id gibi yolları {id: string} nesnesine çevirir.
